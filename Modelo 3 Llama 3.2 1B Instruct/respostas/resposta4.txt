### Tecnologias Principais utilizadas:

1. **Node.js**: O núcleo do sistema, usado para executar os scripts e funções do backend.
2. **Express**: Uma biblioteca para criar aplicativos web e APIs RESTful.
3. **Prisma**: Um ORM (Object-Relational Mapping) para trabalhar com bancos de dados relacionais.
4. **React**: Para criar a interface do usuário (UI), usando componentes reutilizáveis.
5. **Tailwind CSS**: Para estilização avançada e responsividade.
6. **PostgreSQL**: O banco de dados principal para armazenamento de dados.
7. **MongoDB**: Usado como uma fonte de dados adicional para armazenamento de documentos.
8. **Redis**: Um cache para melhorar a performance.
9. **Docker**: Para automatizar a criação e implantação de containers.
10. **AWS Lambda**: Para hospedar funções lambda na AWS.
11. **Azure Functions**: Para hospedar funções Azure Function na Azure.
12. **Google Cloud Functions**: Para hospedar funções Google Cloud Function na Google Cloud.
13. **Heroku**: Para hospedar aplicativos Web na Heroku.
14. **Netlify**: Para hospedar aplicativos Web na Netlify.
15. **GitHub Actions**: Para automatizar tarefas de build e testes.
16. **CI/CD**: Utilizado para automatizar o ciclo de vida do software.
